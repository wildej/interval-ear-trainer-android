# Product backlog

## Идеи (без срока)

- **Игровой режим:** на экране два столбца. В левом — несколько кнопок «прослушать интервал» (желательно с пиктограммой пластинки). В правом — столько же кнопок с названиями интервалов, порядок справа случайный. Пользователь по очереди сопоставляет: прослушал слева — выбрал ответ справа. При ошибке — сигнал; правый столбец можно сменить, левую кнопку считаем уже нажатой.

- **Маскот** (картинка есть) — реакция на верные/неверные ответы; опционально «серия» 3/5/10 угаданных подряд.
